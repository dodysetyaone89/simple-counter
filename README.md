# 🔢 Simple Counter

A clean and modern web-based counter application built using **HTML**, **CSS**, and **JavaScript**. Designed with a minimalist dark interface, this application allows users to increment, decrement, and reset the counter with a smooth and intuitive user experience.

🌐 **Live Demo**  
https://dodysetyaone89.github.io/simple-counter/

## ✨ Features

- ➕ Increment counter
- ➖ Decrement counter
- 🔄 Reset counter
- 💫 Smooth animations
- 🎨 Modern dark UI
- 📱 Responsive layout
- ⚡ Lightweight and fast
- 🚫 No external libraries

## 🚀 Getting Started

Clone this repository

```bash
git clone https://github.com/dodysetyaone89/simple-counter.git
```

Open the project folder

```bash
cd simple-counter
```

Run the application by opening:

```text
counter.html
```

or visit the live demo:

👉 https://dodysetyaone89.github.io/simple-counter/
